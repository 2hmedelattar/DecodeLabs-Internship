# Rule-Based AI Chatbot - Project 1

## Description
This is a simple rule-based AI chatbot built with Python. It is my first project for the DecodeLabs Industrial Training. The chatbot uses basic if-else logic to answer user messages and runs in a continuous loop.

## Features
* **Greetings:** It can say hello and welcome the user.
* **Time:** It can tell you the current time.
* **Exit:** It can say goodbye and close the program.
* **Continuous Loop:** You can keep talking to it until you decide to exit.

## How to Run the Project
1. Make sure you have Python installed on your computer.
2. Download the `Rule_Based_AI_Chatbot.py` file.
3. Open your terminal or command prompt.
4. Run this command:
   `python Rule_Based_AI_Chatbot.py`
