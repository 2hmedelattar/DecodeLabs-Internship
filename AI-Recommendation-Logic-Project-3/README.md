# AI Recommendation Logic - Project 3

## Description
This is my third project for the DecodeLabs Industrial Training. In this project, I built a simple Content-Based Recommendation System using Python. The system takes a user's interests (as text) and matches them against a dataset of items using TF-IDF Vectorization and Cosine Similarity to recommend the best matches.

## Features
* **Text Processing:** Converts user input and item tags into numerical vectors using TF-IDF.
* **Similarity Matching:** Calculates the Cosine Similarity between the user's interests and available items.
* **Smart Ranking:** Sorts and displays the top recommendations based on a percentage match score.

## How to Run the Project
1. Make sure you have Python installed on your computer.
2. Install the required libraries by running this command in your terminal:
   `pip install pandas scikit-learn`
3. Download the `main.py` file.
4. Open your terminal or command prompt.
5. Run this command:
   `python main.py`
